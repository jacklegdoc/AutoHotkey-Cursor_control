# AutoHotkey-Cursor_control

This script was edited with reference to the following link.
https://gist.github.com/toriwasa/64fc9e1a8cce620b8ff92f0ae38075f7

Install ChangeKey and replace CapsLock key to F24 key as administrator according to following link.
http://did2.blog64.fc2.com/blog-entry-349.html

This script can move mouse cursor and click by keyboard.
While CapsLock key is pressed, mouse cursor moves by arrow keys and h, j, k and l keys.
Left Ctrl and Shift keys slower the cursor move.
CapsLock and a combination send left click (CapsLock key release is needed), CapsLock and s send right click, and CapsLock and w send middle click.
CapsLock and m combination moves mouse cursor to right side of active window.
